Detailed Description

Provides GSM communication functionalities for STM32 microcontrollers.

The GSM_Module class encapsulates the functionalities required to interface with a GSM module, such as making calls, sending/receiving SMS, retrieving date and time, checking signal strength and connecting to GPS. It uses UART for communication and processes responses using callbacks.

◆ HAL_UART_RxCpltCallback

void HAL_UART_RxCpltCallback ( UART_HandleTypeDef * huart)
friend

Handles UART interrupts to process incoming commands.

This function is triggered by the IRQ handler when a UART interrupt occurs. It monitors the reception of data to determine if a complete command has been received:

  • If a complete command is received, it transitions to the processing stage.
  • If the command is incomplete, it continues to receive additional data.
